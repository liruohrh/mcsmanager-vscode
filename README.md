# 注意事项

-   取消选中一个 View 的 Item：ESC
-   重新打开工作目录时打开的文件可能显示打开失败，等文件列表获取后再点击重新加载即可
-   批量移动、删除文件会一个一个执行，非常不好，不推荐在工作区批量移动、删除文件
